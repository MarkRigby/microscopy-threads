Alternatives to the Ti/Ti2 pad in NIS Elements

The Ti2 pad can take up a lot of screen are in NIS-Elements and users generally only use it to select the objective and turn on the perfect focus system. Here we show an alternative that provides more useful information to the user and takes up less screen space.

Mark Rigby (Nikon Insturments UK)https://www.microscope.healthcare.nikon.com/
2020-05-18

The Ti2 pad has alot of information, objectives, filter cubes, shutters, diascopic light intensity, etc. However, typically users will encode the majority of this information within an optical configuration and not interact with the Ti2, expect perhaps to change the objective. This post provides an alternative solution.

Firstly, set up the macro panel to execute commands that change the nosepiece position

Next open the system information panel from visualisation controls, and dock below the macro panel. System information can be tailored to

The system information control window indicates the system state. You can display information about the session time, experiment time elapsed, X, Y, and Z position. I personally find having the absolute z-poisition displayed on the y-axis versus time to be extremely useful, though many other variables could be displayed instead or at the same time.

The final result looks alot cleaner, and contains more useful information to the user.
